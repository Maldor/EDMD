# EDMD GUI on Windows

> **This guide is retired.**
>
> The Windows `.exe` installer now handles everything — MSYS2, GTK4, Python, all
> dependencies, and Start Menu shortcuts for both GUI and terminal mode. There is no
> longer a supported manual installation path for Windows.
>
> **→ [WINDOWS_INSTALLER.md](WINDOWS_INSTALLER.md)**

Download `EDMD-Setup-{version}.exe` from the
[latest release](https://github.com/drworman/EDMD/releases/latest) and follow the
instructions in [WINDOWS_INSTALLER.md](WINDOWS_INSTALLER.md).
